# ❤️ NLW IA

![preview](./.github/preview.png)

Projeto construído no evento Next Level Week da Rocketseat.

Aplicação para gerar insights a partir de prompts e aúdios transcritos a partir de vídeos usando IA.

## 🛠️ Tecnlogias

### 🖥️ Front-End
- TypeScript
- React
- Vite
- Tailwind
- Shadcn/ui

### ⚙️ Back-End
- TypeScript
- Fastify
- Prisma
- OpenAi API
